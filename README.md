# テスト駆動で学ぶFirestoreセキュリティルール【CI編】
 
## パッケージインストール
```bash
npm install
```

## テストの実行
### Firestoreエミュレータの起動
```bash
npm run emu:start:firestore
```

### テストの実施
```bash
npm run test-watch
```
